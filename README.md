# 开发环境
- Mac Pro 10.10.1
- Mac Terminal 终端
- Sublime Text 代码编辑器
- Robomongo DongoDB 的 GUI 界面
- Github for Mac 代码管理
- Chrome


## 安装 Meteor和安装包
`$ curl https://install.meteor.com/ | sh`
`$ meteor create demo`
`$ cd demo`
`$ meteor`

- 安装 coffeescript 包
`$ meteor add coffeescript`

- 安装 less
`$ meteor add less`

- 安装 android SDK
`$ meteor install-sdk android`

- Hybrid 工具
`$ meteor add-platform andorid` 
`$ meteor add-platform ios`

- 运行 android 应用
`$ meteor run andorid`
